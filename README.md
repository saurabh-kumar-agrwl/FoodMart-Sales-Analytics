# FoodMart Sales Analytics Dashboard – Excel (Power Query, Power Pivot & DAX)

## Project Overview
This project demonstrates **end-to-end data analytics and reporting capabilities** using advanced Microsoft Excel tools, including **Power Query, Data Modeling, Power Pivot, and DAX**.

The goal was to build an **automated, scalable, and high-performance sales analytics solution** for the FoodMart retail dataset, enabling deeper insights into **transactions, revenue, and profitability**.

---

## Data Preparation & Automation (Power Query)

- Extracted and consolidated **raw transactional data from multiple sources** using Power Query  
- Performed **data cleaning, transformation, and shaping** (handling null values, correcting data types, and standardizing fields)  
- Built **automated refresh pipelines**, allowing reports to update with minimal manual intervention  
- Optimized queries for **performance and reusability**

---

## Data Modeling & Relationships

- Designed a **relational data model** following database normalization principles  
- Created relationships between **fact and dimension tables** (Products, Customers, Brands, Dates, etc.)  
- Implemented **proper cardinality, hierarchies, and star schema design**  
- Leveraged Excel’s **Data Model** to efficiently handle large datasets

---

## Analytics & Measures (Power Pivot & DAX)

Developed **dynamic analytical measures** using **DAX** for advanced performance tracking and time-based calculations.

### Time Intelligence Measures
- **MTD (Month-to-Date)**
- **QTD (Quarter-to-Date)**
- **YTD (Year-to-Date)**

These measures were applied to **daily transaction analysis** to evaluate sales performance across different time periods.

### Key DAX Functions Used
- `CALCULATE`
- `FILTER`
- `SUMX`
- **Time Intelligence Functions**

---

## Key Insights & Visual Analysis

The dashboard delivers actionable insights through several analytical views:

- **Daily Transactions Analysis** with MTD, QTD, and YTD comparisons  
- **Revenue by Product** to identify top-performing and underperforming products  
- **Profit by Brand** to evaluate brand-level profitability  
- **Transactions by Age Group** to understand customer demographics and buying behavior  
- **Transactions by Price Range** to analyze purchasing patterns across price segments

---

## Outcome & Learnings

- Built a **highly optimized Excel-based BI solution** capable of handling large volumes of data  
- Gained practical experience in **data modeling, DAX calculations, and performance optimization**  
- Demonstrated how **Excel can function as a lightweight business intelligence tool**, similar to Power BI  
- Delivered a **decision-ready dashboard** suitable for **sales, marketing, and business stakeholders**
